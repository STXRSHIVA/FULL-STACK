1. HTML:
      2. Defines the root element of an HTML page. All other HTML elements are nested within this
  2. HEAD:
      3. Contains meta-information about the HTML document, such as title, character encoding, links to stylesheets, and more.
      Typically includes the "TITLE" tag
  3. TITLE:
      4. Sets the title of the HTML document, which appears in the browser's title bar or tab.
      Located within the "head" section.
  4. BODY
      5. Contains the content of the HTML document, including text, images, links, and other elements.
  5. H1,H2,H3,H4,H5,H6
      6. Headings ranging from the most important "H1" to the least important "H6".
      Used to structure and organize content hierarchically.
  6. P
      7. Defines a paragraph of text.
      Commonly used for blocks of textual content.
  7. A
      8. Creates a hyperlink to another webpage or resource.
      Utilizes the href attribute to specify the URL.
  8. UL and OL
      9. creates an unordered (bulleted) list.
      creates an ordered (numbered) list.
      Both use "li" for list item
  9. TABLE
      10. Defines a table in the HTML document.
      Uses "tr" for table rows, "td" for table data cells, and "th" for table headers.
  10. DIV and SPAM
      11. "div" is a block-level container used for grouping and styling elements.
      "span" is an inline container used for applying styles or scripting to a portion of text.

    These are some of the essential HTML tags used to structure and present content on a webpage.
